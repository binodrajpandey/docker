Docker Hub is a registry service on the cloud that allows you to download Docker images that are built by other 
communities. You can also upload your own Docker built images to Docker hub. 

1. docker pull jenkins //pulls jenkins image from public registry(Hub)

2. docker run -it centos bash //open bash of centos, you can use cat /etc/*release* to verify the system
use exit command to exit from centos system.

docker images //list all the docker images installed on the system
docker images -q // lists ids only of images
docker rmi imageId //remove docker image
docker inspect jenkins / see the details of jenkins images

docker ps -a //list all the containers in the system, -a tells all lhe containers (exited too) on the system
docker ps //list all the container currently running on the system
docker history imageid // see all the command that we gave against given image
eg. docker history centos

docker stop containerId //stop docker container
docker rm containerId	//delete container
docker stats containerId //provide the statistics of container like cpu and memory utilization of the container
docker attach containerId //attach container to the docker
docker pause containerId //pause the processes in the container
dokcer unpause containerId
docker kill containerId /Kill the process in docker container

In order to build your own image you have to create docker file with instructions on how to build your images.
docker build -t imagename:tagname dir
